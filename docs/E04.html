E04 Production

Device / Branch Not Found

The device or branch ID in the request does not exist in KRA's system.

Likely Causes
  • bhfId not registered for this TIN
  • Device registered under a different branch than specified in the request
Fix
Verify your bhfId against what was assigned during OSCU/VSCU registration. The default sandbox bhfId is '00'. Check that tin + bhfId combination matches your registration.
Gotcha
This is a letter-prefixed code absent from all official spec documents. Only confirmed through live GavaConnect response observation.
