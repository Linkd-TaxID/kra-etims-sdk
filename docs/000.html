000 Official

Success

The request was processed successfully. This is the OSCU spec code per §4.18.

Fix
No action needed.
Gotcha
KRA emits three different success codes depending on integration path: "00" from the VSCU JAR, "000" from OSCU, and "0000" from GavaConnect. Never hardcode a single string. Always check membership in {"0", "00", "000", "0000"}.
